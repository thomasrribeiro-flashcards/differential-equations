# differential-equations

Flashcards for Ordinary Differential Equations — for spaced repetition review via [flashcards](https://github.com/thomasrribeiro/flashcards).

## Coverage

| File | Topic |
|------|-------|
| 01_introduction | ODE definition, order, linearity, IVP/BVP, slope fields |
| 02_first_order_separable_linear | Separation of variables, integrating factor |
| 03_exact_substitutions | Exact equations, homogeneous substitution, Bernoulli |
| 04_first_order_applications | Cooling, mixing, population, RC/RL circuits |
| 05_existence_uniqueness | Picard-Lindelöf, Lipschitz, Picard iteration |
| 06_second_order_homogeneous | Characteristic equation, three root cases, Wronskian, Cauchy-Euler |
| 07_nonhomogeneous_second_order | Undetermined coefficients, variation of parameters |
| 08_second_order_applications | Mass-spring, damping, resonance, RLC circuits |
| 09_higher_order_linear | $n$-th order constant-coefficient ODEs, Wronskian |
| 10_laplace_transforms | Laplace transform, properties, Heaviside, Dirac, convolution |
| 11_laplace_for_odes | Solving IVPs, partial fractions, discontinuous/impulsive forcing |
| 12_systems_of_odes | Eigenvalue method, phase portraits, matrix exponential |
| 13_series_solutions | Power series, ordinary/singular points, Frobenius |
| 14_nonlinear_stability | Autonomous systems, linearization, Lyapunov, phase plane |
